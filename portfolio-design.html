Fakebook (SQL, Java)

EECS 484 class project. Designed a relational database to store information for the fictional social media platform Fakebook. Created an ER Diagram as well as a series of SQL scripts to create, load, and drop objects. Utilized JDBC by creating a Java application that executes several SQL queries against a relational database and places the results in special data structures.

Assembler, Cache + Pipeline Simulator (C)

EECS 370 class project coded in C. Created an assembler to convert code from LC2K to machine code, then a simulator to carry out its instructions. Designed a pipeline simulator that handles data + control hazards. Created a cache simulator that does writeback on store.

Traveling Salesmen (C++)

Designed a program using branch and bound algorithms to solve the traveling salesman permutation problem. Created a dense graph and generated the Minimum Spanning tree using Prim's Algorithm to find the shortest path
